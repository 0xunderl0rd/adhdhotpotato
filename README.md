# ADHD Hot Potato

Static GitHub Pages site for `adhdhotpotato.com`.

## DNS

Configure Namecheap DNS with:

- `A` record, host `@`, value `185.199.108.153`
- `A` record, host `@`, value `185.199.109.153`
- `A` record, host `@`, value `185.199.110.153`
- `A` record, host `@`, value `185.199.111.153`
- `CNAME` record, host `www`, value `0xunderl0rd.github.io`

Do not use a Namecheap URL redirect record for `@`.
